# assets/icons について

素材アイコンの置き場所です。アプリは各素材を次の順で表示します。

1. `assets/icons/<素材名>.png` があればそれを表示
2. 無ければレア度色（緑/青/紫）のプレースホルダを自動生成

`<素材名>` はゲーム内表記そのまま（ファイル禁則文字 `\\ / : * ? " < > |` は `_` に置換）。
例: `海の涙.png`, `冷たいデザート(特製).png`

## アイコンの入れ方

### A. 手動で置く
PNG ファイルを上のファイル名で `assets/icons/` に置くだけ。

### B. URL から一括取得
`assets/icon_sources.json`（素材名 → 画像URL の対応表）に URL を記入し、
リポジトリ直下で次を実行:

```sh
python tools/fetch_icons.py
```

> Tkinter が表示できるのは PNG / GIF です。それ以外は事前に PNG へ変換してください。

## 権利と再配布について

Wiki やゲームから取得した画像は運営（NetEase / Hotta Studio 等）の著作物の
可能性があります。**個人利用に留めてください。**

- `assets/icons/*.png` は `.gitignore` 済み → リポジトリには画像を同梱しません
- 再配布する場合は `assets/icons` を空にすれば、アプリは自動でプレースホルダに
  フォールバックします（コードは画像に依存しません）
- 権利的に問題ない自作画像へ差し替えるのも自由です
